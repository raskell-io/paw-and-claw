# Water Terrain Prompts

## Creek (Shallow Water)

### Base Prompt
```
pixel art shallow creek, top-down tactical game, clear blue water, visible pebble bottom, gentle ripples, forest stream, 32x32 sprite, transparent background
```

### Variations

**Forest Stream**
```
pixel art forest stream, top-down view, babbling brook, rocks in water, sunlight reflections, woodland setting, 32x32 sprite, transparent background
```

**Rocky Creek**
```
pixel art rocky creek, top-down tactical game, water flowing over stones, white foam splashes, mountain stream, 32x32 sprite, transparent background
```

### Animation
```
pixel art creek water, top-down view, flowing water animation, 3 frame sprite sheet, subtle ripple movement, 32x32 per frame, transparent background
```

## Water Edge Tiles

For seamless water edges:
```
pixel art water edge tile, top-down tactical game, [water type] meeting [land type], smooth transition, seamless tiling, 32x32 sprite, transparent background
```

Examples:
- "creek meeting grass"
- "pond meeting forest floor"
- "shore meeting sand"
